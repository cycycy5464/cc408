### 爬虫架构

```
crawler.js → HTTPS GET → HTML解析
  ├─ meta[name=description] → 选择题答案速对
  ├─ explanation-choice divs → 选择题解析
  └─ solution-answer divs → 综合题解析
  ↓ JSON
crawled_data/{year}.json
  ↓
filler.js → 逐行匹配题号 → 填充各年份 .md 文件
```
