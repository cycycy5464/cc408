CPU 组成与层次结构

CPU 基本结构

CPU 主要由以下部分组成：

  • 运算器：执行算术和逻辑运算
  • 控制器：控制指令的执行流程
  • 寄存器组：高速存储单元
  • 内部总线：连接各部件

指令系统

指令格式

| 操作码 (OP) | 地址码 (A1, A2, ...) |
|-------------|---------------------|
| n bits      | m bits each         |

寻址方式

寻址方式 说明 示例
立即寻址 操作数在指令中 MOV R1, #5
直接寻址 指令给出操作数的地址 MOV R1, [1000H]
间接寻址 指令给出操作数地址的地址 MOV R1, @(1000H)
寄存器寻址 操作数在寄存器中 MOV R1, R2
基址寻址 基址寄存器 + 位移量 [BR + offset]
变址寻址 变址寄存器 + 位移量 [IR + offset]

流水线技术

流水线周期

$$T = \max(T_1, T_2, …, T_k)$$

加速比

$$SP = \frac{T_{serial}}{T_{pipeline}} = \frac{n \times k \times T}{n \times T + (k-1) \times T}$$

当 $n \gg k$ 时，$SP \approx k$

存储系统层次

寄存器 → 高速缓存(Cache) → 主存 → 磁盘

Cache 映射方式

  1. 直接映射：每个主存块只能映射到特定的 Cache 行
  2. 全相联映射：主存块可放到 Cache 任意位置
  3. 组相联映射：介于两者之间
