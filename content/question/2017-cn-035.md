在下图所示的网络中，若主机H 发送一个封装访问Internet 的IP 分组的IEEE 802\.11数据帧F,   则帧 F的地址1、地址2和地址3分别是\(\)。

> ![](/cc408/images/questions/2017_Q35_4.svg)

F                                    00\-12\-34\-56\-78\-9c


00\-12\-34\-56\-78\-9a                         00\-12\-34\-56\-78\-9b

A\.00\-12\-34\-56\-78\-9a,00\-12\-34\-56\-78\-9b,00\-12\-34\-56\-78\-9c

B\.00\-12\-34\-56\-78\-9b,00\-12\-34\-56\-78\-9a,00\-12\-34\-56\-78\-9c

C\.00\-12\-34\-56\-78\-9b,00\-12\-34\-56\-78\-9c,00\-12\-34\-56\-78\-9a

D\.00\-12\-34\-56\-78\-9a,00\-12\-34\-56\-78\-9c,00\-12\-34\-56\-78\-9b

[tag_link]

正确答案：B
> IEEE 802.11 数据帧有四种子类型，分别是 IBSS、From AP、ToAP、WDS。
> 这里的数据帧 F 是从笔记本电脑发送往访问接入点（AP），所以属于 To AP 子类型。
> 这种帧地址 l 是 RA (BSSID)，地址 2 是 SA，地址 3 是 DA。
> RA 是 Receiver Address 的缩写，BSSID 是 basic service set identifier 的缩写，SA 是 source address 的缩写，DA 是 destination address 的缩写。
> 因此地址 1 是 AP 的 MAC，地址 2 是 H 的 MAC，地址 3 是 R 的 MAC，选 B。
