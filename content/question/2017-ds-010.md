在内部排序时，若选择了归并排序而没有选择插入排序，则可能的理由是\(\)。

I\. 归并排序的程序代码更短

II\. 归并排序的占用空间更少

II\.   归并排序的运行效率更高

A\. 仅I

B\. 仅 Ⅲ

C\.  仅 I 、Ⅱ

D\. 仅I 、Ⅲ

[tag_link]

正确答案：B
> 参考 排序算法复杂度 ，归并排序代码比选择插入排序更复杂，前者空间复杂度是 O ( n ) , 后者是 O ( 1 ) 。
> 但是前者时间复杂度是 O ( n l o g n ) , 后者是 O ( n 2 ) 。
> 所以 B 正确。
