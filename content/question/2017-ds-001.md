下列函数的时间复杂度是\(\)。

int     func\(int      n\) \{

int    i=     0,   sum = 0;

while   \(sum<n\)sum          \+=\+\+i;

return   i;

A\.O\(log₂n\)

B\.O\(n¹2\)

C\.O\(n\)

D\.O\(nlog₂n\)

[tag_link]

正确答案：B
> sum += ++i; 相当于 ++i; sum = sum + i; 进行到第 k 趟循环， sum = (1+k)*k/2 。
> 显然需要进行 O ( n 1/2 ) 趟循环，因此这也是该函数的时间复杂度。
