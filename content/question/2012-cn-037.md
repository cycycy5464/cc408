下列关于 IP 路由器功能的描述中，正确的是（ ）。

I. 运行路由协议，设置路由表

II. 监测到拥塞时，合理丢弃 IP 分组

III. 对收到的 IP 分组头进行差错校验，确保传输的 IP 分组不丢失

IV. 根据收到的 IP 分组的目的 IP 地址，将其转发到合适的输出线路上

[路由器](/study_methods/tags/408quiz//#%e8%b7%af%e7%94%b1%e5%99%a8)

A\. 仅 III、IV

B\. 仅 I、II、III

C\. 仅 I、II、IV

D\. I、II、III、IV

[tag_link]

正确答案：C

I 和 V 显然是 IP 路由器的
[功能](../../../computer_network/network/dev/content.md#%e5%8a%9f%e8%83%bd)
。对于Ⅱ，当路由器监测到拥塞时，可合理丢弃 IP 分组，并向发出该 IP 分组的源主机发送一个源点抑制的 ICMP 报文。对于 III，路由器对收到的 IP 分组首部进行差错检验，丢弃有差错首部的报文，但不保证 IP 分组不丢失。对于 IV，这表述的也就是路由器的路由功能。
