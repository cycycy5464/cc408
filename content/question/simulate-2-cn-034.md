设信道带宽为 4KHz，信噪比为 30dB，按照香农定理，信道的最大数据速率约等于（ ）。

A\. 10 kb/s

B\. 20 kb/s

C\. 30 kb/s

D\. 40 kb/s

[tag_link]

正确答案：D
> 根据香农定理，信道的最大数据速率由公式 C = B lo g 2 ​ ( 1 + S / N ) 决定，其中 C 为最大数据速率（单位 b/s）， B 为信道带宽（单位 Hz）， S / N 为信噪比线性值。
> 题目给出带宽 B = 4 kHz = 4000 Hz ，信噪比为 30 dB 。
> 首先需要将分贝形式的信噪比转换为线性值，转换公式为 SNR(dB) = 10 lo g 10 ​ ( S / N ) 代入 30 dB 得 30 = 10 lo g 10 ​ ( S / N ) 计算得 lo g 10 ​ ( S / N ) = 3 因此 S / N = 1 0 3 = 1000 接下来代入香农公式： C = 4000 × lo g 2 ​ ( 1 + 1000 ) = 4000 × lo g 2 ​ ( 1001 ) 由于 1001 接近 1000 ，可近似为 lo g 2 ​ ( 1000 ) 。
> 使用换底公式计算 lo g 2 ​ ( 1000 ) = lo g 10 ​ ( 2 ) lo g 10 ​ ( 1000 ) ​ = 0.3010 3 ​ ≈ 9.966 因此 C ≈ 4000 × 9.966 = 39864 b/s 约合 39.86 kb/s 。
> 四舍五入后最接近 40 kb/s ，故选项 D 正确。
