下列有关 RAM 和 ROM 的叙述中，正确的是（ ）。

I. RAM 是易失性存储器，ROM 是非易失性存储器

II. RAM 和 ROM 都采用随机存取方式进行信息访问

III. RAM 和 ROM 都可用作 Cache

IV. RAM 和 ROM 都需要进行刷新

[存储器概念](/study_methods/tags/408quiz//#%e5%ad%98%e5%82%a8%e5%99%a8%e6%a6%82%e5%bf%b5)
[tag_link]
正确答案：A
[RAM](../../../constitution_principle/storage/defs/content.md#ram)
（分为 DRAM 和 SRAM）断电后会失去信息，而 ROM 断电后不会丢失信息，它们都采用随机存取方式（注意，采用随机存取方式的存储器并不一定就是随机存储器）。Cache 一般采用高速的 SRAM 制成，而 ROM 只可读，不能用作 Cache，II 错误。DRAM 需要定期刷新，而 ROM 不需要刷新，故 IV 错误。
