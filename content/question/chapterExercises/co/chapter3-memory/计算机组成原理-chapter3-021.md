假设一个 Cache中共有M 块，每K 块组成一个组，则下列描述中正确的是()。

A. 若 K=1,     则该 Cache 是直接映射 Cache
B. 若 K=1,     则该 Cache 是全相联映射Cache
C. 若 K=M,    则该 Cache 是直接映射 Cache
D. 若 K>1   且K<M,     则该 Cache 是 MIK路组相联映射Cache

[tag_link]

正确答案：A
