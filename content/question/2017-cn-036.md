下列IP 地址中，只能作为 IP 分组的源IP 地址但不能作为目的IP 地址的是\(\)。

A\.0\.0\.0\.0

B\.127\.0\.0\.1

C\.20\.10\.10\.3

D\.255\.255\.255\.255

[tag_link]

正确答案：A
> 0.0.0.0/32 可以作为本主机在本网络上的源地址。
> 127.0.0.1 是 loopback 地址，以它为目的 IP 地址的数据将被立即返回到本机。
> 200.10.10.3 是 C 类 IP 地址。
> 255.255.255.255 是广播地址。
