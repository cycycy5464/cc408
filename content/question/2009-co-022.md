下列选项中，能引起外部中断的事件是\(\)。

A\.  键盘输入

B\.  除数为0

C\.  浮点运算下溢

D\.  访存缺页

[tag_link]

正确答案：A
> 外部中断 指的是 CPU 执行指令以外的事件产生的中断，通常是指来自 CPU 与内存以外的中断。
> A 中键盘输入属于外部事件，每次键盘输入 CPU 都需要执行中断以读入输入数据，所以能引起外部中断。
> B 中除数为 0 属于异常，也就是内中断，发生在 CPU 内部。
> C 中浮点运算下溢将按机器零处理，不会产生中断。
> 而 D 访存缺页属于 CPU 执行指令时产生的中断，也不属于外部中断。
> 所以能产生外部中断的只能是输入设备键盘。
> 操作系统 23 单处理机系统中，可并行的是（）。
> Ⅰ. 进程与进程 Ⅱ. 处理机与设备 Ⅲ. 处理机与通道 Ⅳ. 设备与设备 操作系统概念 A. Ⅰ、Ⅱ、Ⅲ B. Ⅰ、Ⅱ、Ⅳ C. Ⅰ、Ⅲ、Ⅳ D. Ⅱ、Ⅲ、Ⅳ 查看答案与解析 收藏 正确答案： 在单处理机系统（不包含多核的情况）中，同一时刻只能有一个进程占用处理机，因此进程之间不能并行执行。
> 通道是独立于 CPU 的控制输入／输出的设备，两者可以并行，显然，设备与设备之间也是可以并行的。
