在一棵度数为4的树 T中，若有20个度为4的结点，10个度为3的结点，1个度为2的结点，10个度 为1的结点，则树 T 的叶结点个数是\(\)。

A\.41

B\.82

C\.113

D\.122

[tag_link]

正确答案：B
> 设树中度为 i（i=0,1,2,3,4）的结点数分别为 N i ​ , 树中结点总数为 N，则树中各结点的度之和等于 N-1，即 N = 1 + N 1 ​ + 2 N 2 ​ + 3 N 3 ​ + 4 N 4 ​ = N 0 ​ + N 1 ​ + N 2 ​ + N 3 ​ + N 4 ​ ，根据题设中的数据，即可得到 N 0 ​ = 82 ,即树 T 的叶结点的个数是 82。
