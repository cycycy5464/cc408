在采用“取指、译码/取数、执行、访存、写回”5 段流水线的处理器中，执行如下指令序列，其中 s0、s1、s2、s3 和 t2 表示寄存器编号。

```
I1: add s2,s1,s0    // R[s2]←R[s1]+R[s0]
I2: load s3,0(t2)   // R[s3]←M[R[t2]+0]
I3: add s2,s2,s3    // R[s2]←R[s2]+R[s3]
I4: store s2,0(t2)  // M[R[t2]+0]←R[s2]
```

下列指令对中，不存在数据冒险的是（ ）。

[流水线冒险](/study_methods/tags/408quiz//#%e6%b5%81%e6%b0%b4%e7%ba%bf%e5%86%92%e9%99%a9)
[tag_link]
正确答案：C

[数据冒险](../../../constitution_principle/cpu/pipeline/content.md#%e6%95%b0%e6%8d%ae%e5%86%92%e9%99%a9) 指在程序中存在必须等前条指令执行完才能执行后一条指令的情况，此时这两条指令即为数据相关。其中 I1 和 I3、I2 和 I3、I3 和 I4 均发生了写后读相关，因此必须等相关的前条指令执行完才能执行后一条指令。只有 I2 和 I4 不存在数据冒险。所以答案选 C。
