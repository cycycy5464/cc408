下列关于 DMA 方式的叙述中，正确的是（ ）。

Ⅰ. DMA 传送前由设备驱动程序设置传送参数

Ⅱ. 数据传送前由 DMA 控制器请求总线使用权

Ⅲ. 数据传送由 DMA 控制器直接控制总线完成

Ⅳ. DMA 传送结束后的处理由中断服务程序完成

[DMA](/study_methods/tags/408quiz//#dma)
[tag_link]
正确答案：D
每类设备都配置一个
[设备驱动程序](../../../operating_system/io_device/devices/content.md#%e8%ae%be%e5%a4%87%e9%a9%b1%e5%8a%a8%e7%a8%8b%e5%ba%8f)
，设备驱动程序向上层用户程序提供一组标准接口，负责实现对设备发出各种具体操作指令，用户程序不能直接和 DMA 打交道。
[DMA](../../../constitution_principle/bus/io/content.md#dma-%e6%96%b9%e5%bc%8f)
的数据传送过程分为预处理、数据传送和后处理 3 个阶段。预处理阶段由 CPU 完成必要的准备工作，数据传送前由 DMA 控制器请求总线使用权；数据传送由 DMA 控制器直接控制总线完成；传送结束后，DMA 控制器向 CPU 发送中断请求，CPU 执行中断服务程序做 DMA 结束处理。
