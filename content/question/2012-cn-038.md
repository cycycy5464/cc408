ARP 协议的功能是（ ）。

[ARP](/study_methods/tags/408quiz//#arp)

A\. 根据 IP 地址查询 MAC 地址

B\. 根据 MAC 地址查询 IP 地址

C\. 根据域名查询 IP 地址

D\. 根据 IP 地址查询域名

[tag_link]

正确答案：A

在实际网络的数据链路层上传送数据时，最终必须使用硬件地址，
[ARP 协议](../../../computer_network/network/arp/content.md)是将网络层的 IP 地址解析为数据链路层的 MAC 地址。
