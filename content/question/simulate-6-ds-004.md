若一棵二叉树中有 24 个叶结点，有 28 个仅有一个孩子的结点，则该二叉树的总结点数为（ ）。

A\. 70

B\. 73

C\. 75

D\. 77

[tag_link]

正确答案：C
> ���解析】 设二叉树中度为 0 、 1 、 2 的结点数分别为 n 0 ​ 、 n 1 ​ 、 n 2 ​ 。
> 已知叶结点数 n 0 ​ = 24 ，仅有一个孩子的结点数 n 1 ​ = 28 。
> 由二叉树的性质： n 0 ​ = n 2 ​ + 1 ，可得 n 2 ​ = n 0 ​ − 1 = 23 。
> 总结点数 n = n 0 ​ + n 1 ​ + n 2 ​ = 24 + 28 + 23 = 75 。
