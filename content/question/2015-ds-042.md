已知含有5个顶点的图G如下图所示。

![2012_Q41_1](/cc408/images/questions/exam/2015/2015_Q42_1.svg)

请回答下列问题：

(1) 写出图G的邻接矩阵A（行、列下标均从0开始）。

(2) 求A2，矩阵A2中位于 0 行 3 列元素值的含义是什么？

(3) 若已知具有n(n≥2)个顶点的图的邻接矩阵为B，则Bm(2≤m≤n)中非零元素的含义是什么？

[邻接矩阵](/study_methods/tags/408quiz//#%e9%82%bb%e6%8e%a5%e7%9f%a9%e9%98%b5)

1）图A的邻接矩阵如下：

|      | 1    | 2    | 3    | 4    | 5    |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 1    | 0    | 1    | 1    | 0    | 1    |
| 2    | 1    | 0    | 0    | 1    | 1    |
| 3    | 1    | 0    | 0    | 1    | 0    |
| 4    | 0    | 1    | 1    | 0    | 1    |
| 5    | 1    | 1    | 0    | 1    | 0    |

2）A2如下：0 行 3 列的元素值 3 表示从顶点 0 到顶点 3 之间长度为 2 的路径共有 3 条。

|      | 1    | 2    | 3    | 4    | 5    |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 1    | 3    | 1    | 0    | 3    | 1    |
| 2    | 1    | 3    | 2    | 1    | 2    |
| 3    | 0    | 2    | 2    | 0    | 2    |
| 4    | 3    | 1    | 0    | 3    | 1    |
| 5    | 1    | 2    | 2    | 1    | 3    |

3）Bm(2≤m≤n) 中位于i行j列（0≤i,j≤n−1）的非零元素的含义是：图中从顶点i到顶点j长度为m的路径条数。
