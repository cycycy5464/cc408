已知带符号整数用补码表示，float 型数据用 IEEE 754 标准表示，假定变量 x 的类型只能是 int 或 float。当 x 的机器数为 C800 0000H 时，x 的值可能是（ ）。

[补码](/study_methods/tags/408quiz//#%e8%a1%a5%e7%a0%81)
[IEEE浮点数表示](/study_methods/tags/408quiz//#ieee%e6%b5%ae%e7%82%b9%e6%95%b0%e8%a1%a8%e7%a4%ba)
[tag_link]
正确答案：A
C800 0000H = 1100 1000 0000 0000 0000 0000 0000 0000
将其转换为对应的 float 或 int：
1. 为 float 型时，尾数隐藏最高位 1 ,数符为 1 表示负数，阶码
10010000=27+24=128+16
,
再减去偏置值 127 得到 17 ,算出 x 值为
−217
。
2. 为 int 型时，带符号补码，为负数，数值部分取反加 1 ,得 011 1000 0000 0000 0000 0000
0000 0000 ,算出 x 值为
−7×227
。
