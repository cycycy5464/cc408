某进程访问页面的序列如下所示。

![2016_Q1_1](/cc408/images/questions/exam/2016/2016_Q29_1.svg)

若工作集的窗口大小为 6，则在 t 时刻的工作集为（ ）。

[驻留集](/study_methods/tags/408quiz//#%e9%a9%bb%e7%95%99%e9%9b%86)

A\. {6, 0, 3, 2}
B\. {2, 3, 0, 4}
C\. {0, 4, 3, 2, 9}
D\. {4, 5, 6, 0, 3, 2}

[tag_link]

正确答案：A

在任一时刻 t，都存在一个
[集合](../../../operating_system/memory/management/content.md#%e9%a9%bb%e7%95%99%e9%9b%86)
，它包含所有最近 k 次（该题窗口大小为 6）内存访问所访问过的页面。这个集合 w(k, t) 就是工作集。该题中最近 6 次访问的页面分别为 6, 0, 3, 2, 3, 2, 再去除重复的页面，形成的工作集为 {6, 0, 3, 2}。
